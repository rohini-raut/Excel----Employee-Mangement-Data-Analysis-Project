# Excel--Employee-Mangement-Data-Analysis-Project
Analyzed employee data including EmpID,JobTitle,Depatment,Business Unit,Gender,Age,HireDate,Annual Salary,Bonus,Country,City,ExitDate.  Performed data cleaning, applied formulas and created Pivot Tables and charts.  Generated department-wise employee reports, salary analysis, and KPIs for HR insights.
# Tools Used
- Microsoft Excel
  - Pivot Tables,Slicer & Charts

## Key Tasks Performed
- Cleaned and preprocessed employee data (handled duplicates, blanks, and errors)
- Applied Excel formulas for calculations and analysis
- Created Pivot Tables and charts - Generated summary reports and KPIs for HR insights
- Identified department-wise employee distribution and salary patterns
- Identified total Employee By department.
- identified Average Salary by Job Title.
- identified Average Bonus by Department.
- identified Employee Hiring Trend(Year-wise).
- identified Employee Distribution By Countery.
- identified Employee By Gender.
- Provided actionable recommendations for HR decision-making
